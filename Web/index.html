Abstracts

KCGS 2021

We present a robotic pen-drawing system that is capable of replicating images on a large scale surface using TSP art. We use the Traveling Salesman Problem (TSP) to convert arbitrary input raster images into robot drawable vector image. Our robotic system relies on an industrial seven-degree-of-freedom (7DoF) manipulator that is placed on top of a holonomic mobile base robot. Our system uses the mobile base to draw the converted vector image on large surfaces. We demonstrate that our system can generate vector images for robotic drawing using weighted Voronoi Stippling and TSP Art to create pen art with high replication rate on large surfaces.
